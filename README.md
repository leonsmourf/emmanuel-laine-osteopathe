## 📋 Configuration

Pour configurer Google Maps :
1. Remplacez `YOUR_API_KEY` dans `config.js` et `index.html`
2. Ajoutez les restrictions de domaine dans Google Cloud Console
3. Vérifiez que la facturation est activée
